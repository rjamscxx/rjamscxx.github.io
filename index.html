I'm Robert James Cabansay (RJ), a full-stack developer based in the Philippines. I design, build, and ship production web apps end-to-end: from UI and database to auth, deploy, and beyond.

My stack is Next.js + Supabase for production SaaS and offline-first PWAs for constraint-driven client needs. Every project is component-driven, Playwright E2E-tested, and CI/CD-deployed. Philippine businesses deserve software that actually holds up.
